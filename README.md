# This is a React-Redux-Node-MongoDB Full Stack Application


## Install
### Step 1: setup client
   - Go to /notepad
   - npm install  ( wait it done )
   - npm start ( start client )
   
### Step 2: setup server
  - Go to /notepad/server
  - npm install ( wait it done )
  - npm start ( start server )
  
### Step 3: See result
  - Open broswer
  - go to localhost:3000
 
## That's it. Enjoy
